# Iris Flower Classification 🌸

This project uses Machine Learning to classify iris flowers based on their features.

## 📊 Dataset
- Iris dataset (Sepal & Petal measurements)

## 🤖 Algorithm
- Logistic Regression

## 📈 Accuracy
- 100%

## 🛠️ Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🚀 Outcome
The model successfully predicts the species of iris flowers with high accuracy.
